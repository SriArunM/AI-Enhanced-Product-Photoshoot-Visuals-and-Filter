# Zero-Shot Object Detection and Image Manipulation

This project combines state-of-the-art models for zero-shot object detection, segmentation, and image inpainting to create a powerful image analysis and manipulation pipeline.

## Features

- Zero-shot object detection using Grounding DINO and YOLO-World
- Segmentation and masking with Segment Anything Model (SAM)
- Image inpainting using Stable Diffusion
